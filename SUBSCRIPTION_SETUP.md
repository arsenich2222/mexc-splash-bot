# Налаштування перевірки підписки на канал

## Проблема
Бот не може перевіряти підписку, бо не має доступу до каналу @mexcsofts

## Рішення

### Крок 1: Додай бота адміністратором в канал

1. Відкрий канал @mexcsofts в Telegram
2. Натисни на назву каналу → Адміністратори
3. Додати адміністратора
4. Знайди бота: `@your_bot_username` 
5. Додай його з правами:
   - ✅ **Керування учасниками** (щоб перевіряти підписки)
   - Решту прав можна вимкнути

### Крок 2: Отримай правильний ID каналу

Поточний ID в коді: `-1002509308697`

Щоб дізнатися правильний ID:

1. Перешли будь-яке повідомлення з каналу боту @userinfobot
2. Він покаже ID каналу
3. Або використай @getidsbot

**Формат ID:**
- Публічний канал: `@mexcsofts`
- Приватний канал: `-1001234567890` (число)

### Крок 3: Оновити код якщо потрібно

Якщо ID неправильний, зміни в splash.py:

```python
REQUIRED_CHANNEL = "@mexcsofts"
REQUIRED_CHANNEL_ID = -1002509308697  # Заміни на правильний
```

### Тестування

Після додавання бота в канал:
1. Відпишись від каналу
2. Запусти бота `/start`
3. Повинно показати повідомлення про підписку
4. Підпишись на канал
5. Натисни "Перевірити підписку"
6. Повинно працювати!

### Обхід для адміна

Адмін (ID: 1049032098) має доступ завжди, навіть без підписки.
